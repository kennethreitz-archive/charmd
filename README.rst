Charmd: Pow for Python
=======================

like pow but for python. no /etc/hosts. No weird ports.


 curl get.char.md | sh

 cd ~/.charmd
 ln -s /path/to/myapp


 Searches for .wsgi file in folder.

optional ``.charmd`` file w/ virtualenv info.


$ charmd up [site]
# defaults to all

$ charmd down [site]
# defaults to all

# charmd restart
